Submission

Please follow instruction below in order to make a submission:

  1. Download test set here: test.csv. This file contains 14,835 word pairs.
  2. Calculate semantic similarity for each pair of words. Save it in the third column called “sim”. Please keep the original file format unchanged i.e. generate an UTF-8 encoded CSV file separated by commas (“,” sign).
  3. You can provide us up to 6 submissions (up to 3 submissions on each task). For instance, you can upload different versions of your algorithm. Each submission is a copy of the test.csv file with the column “sim” being filled. Please note, that you should fill only one file (test.csv) for both relatedness and association tasks.
  4. Upload files with the results to the form below.
